## mediathek-logo.bin-builder

<p>CLI to unpack/repack the 'logo.bin'-file, that comes with the stock-ROM</p>

---

#### Dependencies

    - gcc (version >= 7.x)
    - zlib (any version)
    - imagemagick (version >= 7.x)
    - ninja (version >= 1.8)

---

#### How to build

<p>Clone the repository, go to the root of it and simply type `ninja`,</p>
